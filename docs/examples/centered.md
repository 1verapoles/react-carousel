## Centered
By default current slide is aligned to the left. You can change that behaviour with centered prop.
```jsx render
<div>
  <Carousel
    centered
    infinite
    arrows
    slidesPerPage={2}
  >
    <img src={imageOne} />
    <img src={imageTwo} />
    <img src={imageThree} />
  </Carousel>
</div>
```
